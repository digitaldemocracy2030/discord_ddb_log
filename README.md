# デジタル民主主義2030 Discord 公開チャットログ・アーカイブ

このリポジトリは、**デジタル民主主義2030** の公式Discordサーバー内で行われた、公開チャンネルおよびパブリックスレッド・フォーラムでの会話ログをオープンデータとして自動蓄積・公開しているアーカイブです。

私たちは、意思決定プロセスの透明化、活動記録の永続化、および後からの情報検索・二次利用を可能にすることを目的として、ログを公開しています。

---

## 📂 データの構造と参照方法

ログデータは `data/` ディレクトリ配下に、以下の2つの形式で保存されています。

### 1. 週別 JSON データ (`data/YYYY-MM-DD_to_YYYY-MM-DD/`)

月曜日始まりの1週間単位でフォルダが分割されており、その中にチャンネル別のJSONファイルが格納されています。

- **通常のチャンネル**: `チャンネル名.json`
- **スレッド・フォーラム**: `親チャンネル名_thread_スレッド名.json`

JSONの各メッセージオブジェクトには、発言内容のほか、チャンネル名、親チャンネル名、タイムスタンプ、添付ファイルURL、リアクション（スタンプ）数などのメタデータが含まれており、人間・LLM・スクリプトのすべてが読み解きやすい形式になっています。

### 2. 年別 SQLite データベース (`data/db/YYYY.db`)

1年ごとに分割された SQLite 3 のデータベースファイルです。
SQLクエリを用いて、過去の全発言から特定のキーワードを横断検索したり、データ分析を行ったりするのに最適です。

## 🔒 プライバシーと免責事項

- **公開対象について**:
  本アーカイブは、サーバーに参加しているメンバー全員が自由に閲覧できる**「公開テキストチャンネル」および「パブリックスレッド / フォーラム」のメッセージのみ**を自動収集しています。メンバー限定のプライベートチャンネル、管理者用チャンネル、およびダイレクトメッセージ（DM）のログは一切収集・公開されません。
- **ログの同期と修正について**:
  Discord上での書き込みは、1日に1回このリポジトリに同期されます。投稿後3日以内に行われたメッセージの「編集」や「削除」は、本アーカイブ側にも自動で同期・反映されます。

## 🧹 過去ログからの個別削除について

過去の自分の投稿について、本アーカイブ（GitHub/SQLite）から個別に削除を希望される場合は、以下の手順および注意事項をご確認の上、Discordサーバーの管理者、または公式サイトの問い合わせ窓口までメッセージIDや該当箇所をご連絡ください。

### ⚠️ 注意事項（必ずご確認ください）

- **Discord上のメッセージの削除**: アーカイブから削除を行っても、Discord上に対象メッセージが残ったままだと、日次の自動同期処理によってアーカイブへ再度取り込まれ、データが復活してしまいます。そのため、**あらかじめご自身で（または管理者の手で）Discord上の対象メッセージを削除した上で**、アーカイブからの削除をご依頼ください。
- **対応にかかる時間**: アーカイブデータの削除対応はボランティアによって手動で行われるため、対応の完了までに数日～数週間程度、場合によってはそれ以上のお時間をいただく可能性があります。あらかじめご了承ください。
